git checkout

git checkout mybranch

Switch to the existing branch myBranch.
This means:
  1. Set HEAD to this branch.
  2. Adjust workspace files to match myBranch.

git switch -c myNew myOld

Create the new branch myNew based on the already existing branch myOld, set HEAD to myNew and adjust workspace accordingly.

git switch -

Switch to the previous branch

git switch @{-3}

Switch to the third latest switch target.

git reflog | findstr "moving from "

git reflog | grep -oE "moving from [^ ]+ to [^ ]+" | awk '{print $4}' | awk '!seen[$0]++' | head -n 10

Show the latest switch targets. The first variant is Windows, the second Linux.
