git revert

git revert 1542affe

Create a new commit that reverts all changes of the commit 1542affe. Build the commit message from the reverted commit, prepending it by the word "Revert".

git revert b1..b2

Revert all the changes contained in branch b2 but not in branch b1. Create a new commit for every reverted commit. Do this reverting in reverse order.

git revert ^b1 b2

Revert the commit b1, then the commit b2. This means that ^b1 b2 is NOT the same as b1..b2 as it is in other cases!
